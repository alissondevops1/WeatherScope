# WeatherScope 🌦️

Modern weather application built with Python, PyQt5 and OpenWeather API.

## Features

- Search weather by city
- Modern dark interface
- Temperature display
- Feels like temperature
- Humidity information
- Wind speed
- Weather emojis
- Error handling
- Refresh button

## Technologies

- Python
- PyQt5
- Requests
- OpenWeather API

## Installation

Clone the repository:

```bash
git clone https://github.com/alissondevops1/WeatherScope.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python main.py
```
